A TeszemVeszem egy online piactér, amely lehetőséget biztosít a felhasználóknak a használt informatikai eszközeik eladására, illetve a mások által meghirdetett tárgyak böngészésére.
A vásárlás-eladási folyamatba a TeszemVeszem harmadik félként nem vállal szerepet. 
A projekten közreműködött:
•	Szarvas Ádám László
•	Skorka Dávid
•	Lazanyecz Márk

Virtuális környezet létrehozásához:
python -m venv .venv

Python verzió:
Python 3.12

A projekt futtatásához:
pip install -r requirements.txt
